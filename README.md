## Desafio Front

Para rodar o projeto:

Primeiro adicionar as dependências:

```yarn```

Rodar a aplicação com o comando:

```yarn start```

Para que a aplicação rode é preciso inserir o token no arquivo .env;

Eu deixei um arquivo .env.examples, é só mudar o nome para .env e inserir o token;

```REACT_APP_TOKEN=<token>```

A aplicação roda em localhost:3000 em modo de desenvolvimento;
